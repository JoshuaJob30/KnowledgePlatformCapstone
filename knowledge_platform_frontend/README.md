Knowledge Platform Frontend

Overview:
This is the Next.js frontend for the Knowledge Platform Capstone. It provides:
Document upload interface (UploadForm).
Query interface (QueryBox).
Answer display with source labels (AnswerDisplay).
Navigation bar and footer for professional layout.
MUI + Tailwind styling.

Requirements:
Node.js 20+
npm or yarn

Setup:
# Clone repo
git clone https://github.com/JoshuaJob30/KnowledgePlatformCapstone.git
cd KNOWLEDGE_PLATFORM_FRONTEND
# Install dependencies
npm install

Run Locally:
npm run dev
